# Mallid

See kaust koondab digiloovtöö läbiviimiseks vajalikud **täidetavad mallid**.

Mall on struktuurne põhi, mida õpilane, meeskond või õpetaja saab oma projekti jaoks täita, kohandada ja täiendada. Mall ei ole iseseisev õppetegevus, vaid aitab hoida digiloovtöö protsessi nähtava, korrastatud ja dokumenteerituna.

Mallid toetavad digiloovtöö kõiki olulisemaid osi:

- probleemi ja kasutajavajaduse kirjeldamist;
- persoonade ja stsenaariumide koostamist;
- projekti planeerimist;
- backlog’i ja Kanbani töövoo loomist;
- sprintide kavandamist;
- prototüübi kirjeldamist;
- testimise ja kasutajatagasiside kogumist;
- dokumentatsiooni koostamist;
- refleksiooni ja retrospektiivi läbiviimist;
- lõppesitluse ja pitch’i ettevalmistamist;
- näidisprojektide ja õppematerjalide ühtses formaadis kirjeldamist.

## Milleks malle kasutada?

Mallid aitavad õpetajal ja õpilastel liikuda digiloovtöö protsessis samm-sammult.

Nende abil saab:

- muuta tööprotsessi läbipaistvamaks;
- vähendada õpetaja juhendamiskoormust;
- anda õpilastele selge struktuuri;
- toetada meeskonnatööd;
- koguda protsessihindamiseks vajalikke tõendusmaterjale;
- siduda töövoo, tagasiside ja refleksiooni;
- hoida erinevate projektide dokumentatsiooni sarnases vormis.
