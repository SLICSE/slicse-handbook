# Kuidas seda käsiraamatut kasutada?

See käsiraamat aitab informaatikaõpetajal kavandada, juhendada ja hinnata digiloovtööd.

## Kui alustad esimest korda

1. Loe: `Digiloovtöö kiirtee õpetajale`
2. Võta kasutusele: `Projektiplaani mall`
3. Jaga õpilastele: `Meeskonna rollikaardid`
4. Kasuta iga nädal: `Sprindi kontrollnimekiri`
5. Hinda protsessi: `Protsessihindamise rubriik`

## Kui vajad hindamist

Alusta jaotisest `07-hindamine`.

## Kui vajad valmis näidet

Alusta jaotisest `08-naidisprojektid`.
