
# Alusta siit

See jaotis aitab õpetajal kiiresti aru saada, kuidas SLICSE käsiraamatut kasutada.

## Kui kasutad käsiraamatut esimest korda

1. Tutvu digiloovtöö üldise ideega.
2. Vali sobiv näidisprojekt.
3. Kasuta projektiplaani malli.
4. Jaga õpilastele rollikaardid.
5. Toeta tööd kontrollnimekirjade abil.
6. Hinda protsessi ja tulemust rubriigi alusel.

## Käsiraamatu põhijaotised

- Juhendid
- Töölehed
- Mallid
- Rollikaardid
- Kontrollnimekirjad
- Rubriigid
- Näidisprojektid

## Kellele see käsiraamat on mõeldud?

Käsiraamat on mõeldud eelkõige eestikeelsetele informaatikaõpetajatele ja digiloovtöö juhendajatele.
