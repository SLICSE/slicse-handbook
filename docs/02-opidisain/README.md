# Õpidisain

See jaotis selgitab digiloovtöö metoodilisi aluseid. Digiloovtöö ei põhine ühelainsal meetodil, vaid ühendab õpidisaini, projektipõhise õppe, disainiuuringu, kasutajakeskse disaini ning agiilse töökorralduse põhimõtteid.

Digiloovtöö kavandamisel annab ADDIE mudel üldise õpidisaini raami, disainiuuring toetab iteratiivset arendus- ja uurimisloogikat ning projektipõhine õpe loob tähendusliku õpikonteksti. Kasutajakeskne disain, persoonad ja stsenaariumid aitavad õppijatel mõista probleemi, kasutajat ja kasutusolukorda. Agile Manifesto, Kanban ja sprindid toetavad nähtavat, juhendatud ja tagasisidestatud töövoogu.

## Mida siit jaotisest leiab?

* õpidisaini üldpõhimõtted;
* ADDIE mudeli rakendamise digiloovtöös;
* disainiuuringu rolli digiloovtöö arendamisel;
* projektipõhise õppe seosed digiloovtööga;
* kasutajakeskse disaini, persoonade ja stsenaariumide kasutamise;
* Agile Manifesto tõlgenduse hariduskontekstis;
* Kanbani ja sprintide kasutamise õppeprotsessi korraldamisel;
* enesejuhitud õppimise rolli agiilses töövoos;
* õpiteekonna kavandamise põhimõtted.
