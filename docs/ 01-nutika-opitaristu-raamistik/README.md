# Nutika õpitaristu raamistik

See jaotis selgitab SLICSE käsiraamatu teoreetilist ja kontseptuaalset lähtekohta: kuidas mõista nutikat õpitaristut digiloovtöö kontekstis ning kuidas nähtav, juhendatud ja andmetega toetatud töövoog aitab õpetajal pikemaajalist projektipõhist õppeprotsessi korraldada.

Nutika õpitaristu all mõistetakse siin õppimist toetavat terviklikku korraldust, kus õpidisain, digitaalsed töövahendid, õpetaja orkestreerimine, õppija enesejuhtimine, tagasiside, hindamine ja tööprotsessi nähtavus moodustavad omavahel seotud süsteemi.

Digiloovtöö puhul ei ole oluline ainult lõpptulemus ehk valminud prototüüp. Sama oluline on nähtav tööprotsess: kuidas õpilased sõnastavad probleemi, mõistavad kasutajat, loovad backlog’i, liiguvad Kanbani töövoos, töötavad sprintides, saavad tagasisidet, dokumenteerivad arengut ja reflekteerivad oma õppimist.

## Mida siit jaotisest leiab?

* nutika õpitaristu üldise ülevaate;
* põhimõisted, mida kasutatakse SLICSE käsiraamatus;
* selgituse, kuidas agiilne töökorraldus toetab õpitaristut;
* Kanbani käsitluse nähtava töövoona;
* sprindi käsitluse õppeprotsessi korduva üksusena;
* õpetaja orkestreerimise rolli nutikas õpitaristus.
