# Uuringud ja publikatsioonid

See kaust koondab SLICSE käsiraamatu teadusliku ja avaliku taustamaterjali: artiklite, konverentsiettekannete, raamatupeatükkide ja blogipublikatsioonide kaardid.

Kaardid ei ole täistekstid ega refereeringud kogu artiklist. Iga kaart aitab õpetajal, koolijuhil, õppedisaineril või uurijal kiiresti mõista:

- millest publikatsioon räägib;
- milline on selle seos digiloovtööga;
- milliseid põhimõtteid saab kasutada SLICSE käsiraamatus;
- kuidas publikatsioon toetab õpetaja juhendamist, õppija toestamist, protsessihindamist või nutika õpitaristu kujundamist;
- milliste käsiraamatu osadega publikatsioon on seotud.

## Kuidas neid kaarte kasutada?

### Õpetaja jaoks

Õpetaja saab artiklikaartidest kiiresti tausta, miks digiloovtöö ei ole lihtsalt uus nimi loovtööle, vaid teistsugune töökorraldus:

- koostöine projekt;
- päriseluline probleem;
- kasutajavajadus;
- prototüüpimine;
- Kanban või muu nähtav töövoog;
- protsessihindamine;
- refleksioon;
- õpetaja juhendav ja orkestreeriv roll.

### Koolijuhi jaoks

Koolijuht saab kaarte kasutada digiloovtöö rakendamise põhjendamiseks kooli tasandil:

- miks loovtöö juhend vajab uuendamist;
- kuidas digiloovtöö seostub uue informaatika õppekavaga;
- millist tuge vajavad õpetajad;
- miks on vaja aega, töövoogu ja selget hindamismudelit;
- kuidas digiloovtöö saab toetada kooli digiuuendust.

### Õppedisaineri ja haridustehnoloogi jaoks

Õppedisainer või haridustehnoloog saab kaarte kasutada SLICSE käsiraamatu materjalide arendamisel:

- milliseid töölehti on vaja;
- milliseid malle on vaja;
- millist õppija toestust tuleb pakkuda;
- kuidas siduda Kanban, sprint, backlog ja refleksioon;
- kuidas kujundada õpetaja töölauda või juhendaja töövoogu.

### Uurija jaoks

Uurija saab kaarte kasutada SLICSE teadusliku tausta ja argumentatsiooni korrastamiseks:

- millised publikatsioonid toetavad digiloovtöö kontseptsiooni;
- millised artiklid kirjeldavad disainiuuringut;
- millised tulemused puudutavad õpetaja rolli;
- millised tulemused puudutavad õppija enesejuhtimist;
- millised tulemused puudutavad platvormi, Taigat, Kanbani ja õppematerjale.

## Põhiteemad kaartides

Artiklikaarte läbivad korduvad teemad:

- digiloovtöö kui koostöine ja kasutajakeskne loovtöö vorm;
- põhikooli informaatika õppekava muutumine;
- arvutusliku mõtlemise ja disainmõtlemise ühendamine;
- agiilne töövoog koolikontekstis;
- Kanban kui nähtav õppimise ja töökorralduse vahend;
- Taiga, GitHub või muu projektikeskkond kui õpitaristu osa;
- õpetaja roll juhendaja ja orkestreerijana;
- õppija järkjärguline toestamine;
- meeskonnatöö ja rollijaotus;
- protsessihindamine;
- refleksioon ja enesehindamine;
- näidisprojektid, mallid, töölehed ja videojuhendid;
- koolijuhtide ja õpetajate valmisolek õppekava uuenduseks.
